# GitHub Repo Finder

A simple web application that helps users discover random GitHub repositories based on a selected programming language.

## Features

- Select a programming language
- Fetch repositories using the GitHub API
- Display a random repository
- Refresh to view another random repository
- View repository directly on GitHub
- Loading indicator during API requests
- Language-specific color indicators

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub REST API

## Future Improvements

- Display repository owner avatar
- Handle missing repository descriptions
- Add repository search filters
- Improve mobile responsiveness
